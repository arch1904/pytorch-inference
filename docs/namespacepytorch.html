Converts a numpy array to an ArrayFire array.

Parameters
arraynumpy ndarray (PyArrayObject *) object
ndimNumber of dimensions, usually 4
dimsThe array dimensions - note these are in the torch convention (n, k, h, w), we will convert to ArrayFire convention which is (h, w, k, n) in the output array. This will be obscured as much as possible.
Returns
ArrayFire array that has the data from the numpy array arranged within.
